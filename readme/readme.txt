Настройка:
  config.txt = ip, port
  У клиента должен был записан ip сервера
  У клиента и сервера должен быть одинаковый порт

Управление:
  Шарик летит к указателю мыши
  Чем дальше указатель, тем больше скорость
  Левый щелчок мыши - деление на части

Цель:
  Нужно есть соперников.
  Чем больше масса, тем ты круче... но медленнее
